## About
This repository contains five different Python projects showcasing various programming techniques and best practices. 

## Project Descriptions

### Calculator Application
A simple and user-friendly calculator application that can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

### Contact Book Application
An application to store and manage personal contacts, allowing users to add, delete, and edit contact information efficiently.

### Password Generator Application
This application generates random strong passwords for users, providing options for length and complexity, thereby enhancing security.

### Rock Paper Scissors Game Application
A fun command-line game where players can compete against the computer by choosing rock, paper, or scissors.

### To-Do List Application
A task management application that lets users create, update, and delete tasks, making it easy to organize daily activities.
